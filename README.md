## Variables

- `APP_ID`  =  Get this value from my.telegram.org
- `API_HASH`  =  Get this value from my.telegram.org
- `SESSION`  =  Get this by using [Repl.it](#Repl) or from [terminal](#Terminal)
- `HANDLER`  =  Your command handler.
- `SUDO_USER`  = Sudo id
